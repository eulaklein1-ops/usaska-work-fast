FAQ

Частые вопросы

Как лиды попадают в Google Таблицу?

Форма отправляет данные в Google Apps Script Web App. В архиве есть готовый файл google-apps-script.js и инструкция по подключению.

Этот сайт можно залить на GitHub Pages?

Да. Сайт полностью статический. Для лидов нужен только ваш Google Apps Script URL, который вы вставляете в js/config.js.

Можно ли менять картинки и тексты?

Да. Все изображения лежат в папке assets, тексты — в index.html. Можно быстро заменить под другой оффер.

Сохраняются ли UTM-метки?

Да. Страница автоматически записывает query-параметры в поле utm и отправляет их вместе с заявкой.
